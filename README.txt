R For Data Scientists Project-1 Readme

1)	Install the following input files from the UTD server:
•	BBC-text.csv (https://www.utdallas.edu/~vxk180030/BBC-text.csv)
•	Wiki-news(https://www.utdallas.edu/~vxk180030/wiki-news-300d-1M.vec )
2)	Now get the path of those downloaded files and then paste it in the space where I have commented in the code.
3)	Now run the code to get the output metrics and predictions for 25 test data points.
4)	To run on g cloud just use mlcloud_train of the filename. The defaults paths given are dataset paths stored in Google storage.
